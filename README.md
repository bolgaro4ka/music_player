# Music Player
A music player with the ability to rewind to a specific point written in Python using Tkinter and Pyglet

## Functions
- Play, pause
- Next, previos music
- Volume
- Rewind
- Random music
- Open music folder

## Imports
 - Pyglet
     - ``` pip install pyglet```
 - Tinytag
     - ``` pip install tinytag```
